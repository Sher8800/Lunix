First level: Коротко дайте ответы на след. вопросы:

  1. Зачем предназначен файл .gitignore?

    2. .gitignore нужен для скрытия файлов и папок от системы контроля версий Git. Например файлы с поролями, системные файлы и тд.

  2. Что делает команда git remote rm origin?

    3. Удоляет связь с удоленным репозиторием.

  3. В каких случаях происходит MERGE CONFLICT?

    4. MERGE CONFLICT происходит обычно тогда когда два разроботчика вносят какие-либо изменения в файл в одном и том же месте.

  4. Как посмотреть отличия между ветками?

    5. git diff ветка1 ветка2

  5. Какой командой удалить локальную и удаленную ветки?

    6. Чтобы удалить локальную ветку мы используем команду git branch -d (или -D для принудительного удоления).
    Для удоления удоленной ветки вводим следующую команду: git push origin -d "name-branch"

  6. Как сделать файл неотслеживаемым (untracked)?

    7. Чтобы сделать файл неотслеживаемым нужна команда: git reset "namefile"
